## 2235.Add Two Integer

Given two integers (num1) and (num2), return the sum of the two integers.

#### My pseudocode:
return (number 1 + number 2)

-----

### C

#### My solution:
```
int sum(int num1, int num2) {
    return(num1 + num2);
}
```

-----

### Python

#### My solution:
```
class Solution(object):
    def sum(self, num1, num2):
        """
        :type num1: int
        :type num2: int
        :rtype: int
        """
        return(num1 + num2)
```
